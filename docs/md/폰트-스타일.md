# 폰트 스타일 정리

## 1. 개요
- IBM Plex Sans KR 임베드 적용함.
- 웹 외부 로드 제거하고 로컬 폰트 사용함.
- 모든 플랫폼 공통 텍스트 스케일 정의함.

## 2. 파일 배치
- 폰트 위치: `assets/fonts/ibm_plex_sans_kr/`
- 사용 파일: Thin/ExtraLight/Light/Regular/Medium/SemiBold/Bold

## 3. 적용 구조
```text
AppTheme -> AppTypography -> TextTheme -> Widgets
```

## 4. pubspec 설정
```yaml
flutter:
  fonts:
    - family: IBM Plex Sans KR
      fonts:
        - asset: assets/fonts/ibm_plex_sans_kr/IBMPlexSansKR-Thin.ttf
          weight: 100
        - asset: assets/fonts/ibm_plex_sans_kr/IBMPlexSansKR-ExtraLight.ttf
          weight: 200
        - asset: assets/fonts/ibm_plex_sans_kr/IBMPlexSansKR-Light.ttf
          weight: 300
        - asset: assets/fonts/ibm_plex_sans_kr/IBMPlexSansKR-Regular.ttf
          weight: 400
        - asset: assets/fonts/ibm_plex_sans_kr/IBMPlexSansKR-Medium.ttf
          weight: 500
        - asset: assets/fonts/ibm_plex_sans_kr/IBMPlexSansKR-SemiBold.ttf
          weight: 600
        - asset: assets/fonts/ibm_plex_sans_kr/IBMPlexSansKR-Bold.ttf
          weight: 700
```

## 5. 테마 설정
```dart
final colorScheme = ColorScheme.fromSeed(
  seedColor: Colors.blue,
  brightness: Brightness.light,
);

ThemeData(
  useMaterial3: true,
  fontFamily: AppTypography.fontFamily,
  colorScheme: colorScheme,
  textTheme: AppTypography.buildTextTheme(colorScheme),
);
```

## 6. 텍스트 스케일
- displayLarge 57/700/1.2
- displayMedium 45/700/1.2
- displaySmall 36/700/1.2
- headlineLarge 32/600/1.25
- headlineMedium 28/600/1.25
- headlineSmall 24/600/1.3
- titleLarge 22/600/1.3
- titleMedium 16/500/1.35
- titleSmall 14/500/1.35
- bodyLarge 16/400/1.45
- bodyMedium 14/400/1.45
- bodySmall 12/400/1.45
- labelLarge 14/600/1.3
- labelMedium 12/600/1.3
- labelSmall 11/600/1.3

## 7. 사용 예시
```dart
Text(
  '안녕하세요',
  style: Theme.of(context).textTheme.titleLarge,
)
```

## 8. 운영 체크
- `web/index.html`의 Google Fonts `<link>` 불필요함.
- 변경 후 `flutter pub get` 실행 필요함.
